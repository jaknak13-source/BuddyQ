# BuddyQ
BuddyQ is a fully local AI voice assistant for Windows that runs on your own hardware. It uses faster‑whisper, Piper TTS, and an OpenAI‑compatible local LLM (LM Studio or llama.cpp) to provide private, always‑on voice control with web search, timers, notes, weather, and screen/vision tools. No cloud, no API keys, no subscriptions!
